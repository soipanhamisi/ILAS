# IntelliJ npm Run Configuration - Visual Guide

## Quick Steps

### Step 1: Open Configurations
```
Click: Run → Edit Configurations...
```

### Step 2: Add New npm Configuration
```
Click: [+] button → Select: npm
```

### Step 3: Fill In Details
```
Name:                Frontend Dev Server
Package.json:        Browse to frontend\package.json
Command:             run
Scripts:             dev
Working directory:   Browse to frontend folder
Run in Terminal:     CHECK this box
```

### Step 4: Save & Run
```
Click: [OK]
Select: Frontend Dev Server from dropdown
Click: Green [▶] Run button
```

### Step 5: Access Frontend
```
Terminal shows: ➜  Local: http://localhost:3000/
Open in browser
```
